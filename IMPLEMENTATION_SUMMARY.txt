# 📊 ALGORITHM VERIFICATION

Anti-Spoofing Algorithm (EXACT):
├─ Input: face_image (BGR)
├─ Step 1: Resize → 256x256 ✓
├─ Step 2: Normalize → pixel / 255.0 ✓
├─ Step 3: Calculate Laplacian
│        └─ Kernel: [0  -1  0]
│                  [-1  4 -1]
│                  [0  -1  0] ✓
├─ Step 4: Laplacian variance < 50 → REJECT ✓
├─ Step 5: FaceAntiSpoofing.tflite inference ✓
├─ Step 6: Score calculation
│        └─ score = sum(abs(pred[i]) * mask[i]) ✓
└─ Step 7: If score < 0.2 → REAL else FAKE ✓

Embedding Algorithm:
├─ Input: face_image (BGR, any size)
├─ Resize → 112x112 ✓
├─ Normalize → (pixel - 0.5) / 0.5 ✓
├─ MobileFaceNet.tflite inference → 128-dim ✓
└─ L2 normalize → embedding ✓

Enrollment Algorithm (5 poses):
├─ Pose 1: Front → extract embedding ✓
├─ Pose 2: Left → extract embedding ✓
├─ Pose 3: Right → extract embedding ✓
├─ Pose 4: Up → extract embedding ✓
├─ Pose 5: Down → extract embedding ✓
├─ Average: final_emb = mean([emb1-5]) ✓
├─ L2 normalize ✓
└─ Store in database ✓

Recognition Algorithm:
├─ Anti-spoofing check
│  ├─ If FAKE → return FAKE ✓
│  └─ If REAL → continue ✓
├─ Extract embedding from face ✓
├─ Cosine similarity vs all users:
│  └─ sim = dot(emb_a, emb_b) / (norm_a * norm_b) ✓
├─ Max similarity among all users ✓
└─ If max_sim >= 0.7 → MATCH else UNKNOWN ✓
